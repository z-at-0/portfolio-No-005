# Portfolio No-005

## 概要

このサイトは、GitHub APIを利用してユーザー情報やリポジトリを検索・表示するツールです。
非同期処理の制御やAPI連携など、フロントエンドにおける技術的なロジックの実装に重点を置いています。

## 目標・こだわり

- **通信の最適化:** `AbortController` を使用し、連続したリクエストによる表示の不整合（競合）を防止
- **効率的なイベント管理:** 「イベント委譲」を採用し、動的に生成されるリストに対してメモリ負荷の低いクリック処理を実装
- **ユーザー体験（UX）の向上:** ローディング状態の表示や、APIの利用制限（Rate Limit）を可視化することで、利用者が状況を把握しやすい設計

## 主な機能

- **GitHubユーザー検索:** アカウント名、アイコン、自己紹介（Bio）をリアルタイム取得
- **リポジトリ管理:** スター数順のソート、および「次へ」ボタンによるページネーション表示
- **詳細プレビュー:** 選択したリポジトリの概要表示、および最新5件のコミット履歴の取得

## 技術スタック

- **使用言語:** HTML, CSS, JavaScript (Vanilla JS)
- **技術要素:** Fetch API, AbortController, GitHub REST API
- **ライブラリ:** なし（標準機能のみで高度な非同期制御を実装）
