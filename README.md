# Portfolio No-005: GitHub User & Repo Explorer

## 概要
GitHub APIを活用し、特定のユーザーのプロフィール、リポジトリ一覧、および最新のコミット履歴をリアルタイムで取得・表示するシングルページアプリケーション（SPA）です。

単なるデータ取得に留まらず、通信の最適化やエラーハンドリングを重視して設計しました。

## 目標・こだわり

### 1. 通信の最適化と整合性（AbortController）
短時間での連続検索やリポジトリ切り替え時に、古いリクエストをキャンセルするために `AbortController` を導入しました。  
これにより、低速なネットワーク環境でも表示内容のズレを防いでいます。

---

### 2. 効率的なイベント管理（イベント委譲）
個々のリポジトリ要素にイベントを付与するのではなく、親要素でクリックを検知する「イベント委譲」を採用しています。  
動的に生成される要素にも対応でき、メモリ効率も改善しています。

---

### 3. API制限への配慮
GitHub APIのレートリミットを画面上に表示し、現在の利用可能回数を可視化しています。

---

## 主な機能

- ユーザー検索（アカウント名・アイコン・Bio取得）
- リポジトリ一覧表示（更新順取得 + スター数ソート）
- リポジトリ詳細表示（スター数・説明・リンク）
- コミット履歴表示（最新5件）
- ページネーション（「次へ」で追加取得）
